# CS2_Map_Creator
Transforms a png of satellite imagery to a grayscale readable by Cities Skylines 2 to create maps.

Program will read in a user input for a png's file destination
Program scales png if size is incorrect (4096x4096)
Png is then greyscaled to proper shades for readibiltiy in Cities Skyline 2 (Brighter == Higher Altitudem | Darker == Lower Altitude)
Created heightmap is placed in output folder
